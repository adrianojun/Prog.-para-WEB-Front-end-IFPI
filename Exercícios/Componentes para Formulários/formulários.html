Componetes Novos

datalist

O componente datalist define um conjunto de opções que podem ser associadas a um campo de entrada de texto. Essas opções são exibidas como sugestões ao usuário enquanto ele digita no campo de entrada. O usuário pode selecionar uma das opções sugeridas ou continuar digitando seu próprio valor. Claro! O componente date é usado para criar um campo de entrada de data em formulários HTML. Ele permite que os usuários selecionem uma data usando um calendário ou digitem diretamente no campo.
