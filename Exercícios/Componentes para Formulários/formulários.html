Compentes Antigos

Radio

O componente radio é usado para criar botões de opção em formulários HTML. Ele permite aos usuários selecionar apenas uma opção em um grupo de opções mutuamente exclusivas.

Possuem alguns atributos importantes como o "value" que indica o botão selecionado. "name" Identifica qual grupo pertence o botão de rádio.

Novos Componentes

Datalist

O datalist é usado em formulários HTML para fornecer uma lista de opções pré-definidas para um campo de entrada de texto. Ele permite que os usuários escolham uma opção da lista ou digitem seu próprio valor.

date

O date é usado em formulários HTML para criar um campo de entrada específico para datas. Ele exibe um calendário para que os usuários possam selecionar uma data facilmente.
